Ninkasi - Bierminen

Ninkasi index

There are two ways to use Ninkasi: query for a single beer style by its number or query for all beer styles.

Query for a beer style

To query for a single beer style, access append the following to URI --change 50 as you wish:

/style/50

Query for all styles

To retrieve all styles, use the following URI paramenter:

/style
